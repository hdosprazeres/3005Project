Setting it up:

Make sure the database is initialized and the server.js file is set up with
the proper database settings:
    Port
    Database name
    Password

Starting it up:
    1.The server should be started through nodemon package. To make sure all
    packages are properly installed, open the terminal in the folder where the
    files are stored and type "npm init"

    2.Start the server by typing "nodemon server.js"

    3.Open "localhost:3000" on your web browser.
